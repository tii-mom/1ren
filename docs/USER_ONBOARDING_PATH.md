# 1人算力终端用户成长与新手指南

本指南旨在向新接入 R1 增长终端的用户，展示如何从一个零基础的“算力体验者”逐步成长为发起项目影子募资的“1人算力公司创始人”。

---

## 阶段一：初入终端与免费体验 (Level 0)

1.  **注册与免费额度**：
    *   用户首次进入终端，可在 Header 免费领取 **500 USDT 模拟测试金**，并自动获赠少量初始 R1 余额。
2.  **激活试用算力**：
    *   在【我的后台】或【设备机房】一键领取并激活 **“本地显卡共享算力体验节点 (7天试用)”**。该模拟设备在不消耗电费和维护费的前提下，会每日为用户产出少量 AI Token (算力碎片)。

---

## 阶段二：算力并网与 API 额度兑换 (Level 1)

1.  **租赁商业设备**：
    *   使用模拟金 USDT，在【设备机房】中租用例如 **“GPU 并网算力节点 S1”**。
2.  **管理设备维护**：
    *   监控算力设备的运行状态。当设备长时间高负荷运转导致超温降频（状态变更为 `decayed`）时，在【服务大厅】使用 AI Token 兑换并使用**温控维护液**，使设备效率恢复 100% 满载运转。
3.  **大模型自用与变现**：
    *   当积累了一定数量的 AI Token 后，用户可选择在【服务大厅】兑换 **API 服务凭证包** 将其提取为可调用的大模型 API Key；或者在首页财富面板通过**一键回收**，将算力碎片折算为 USDT。

---

## 阶段三：团队节点裂变与市场套利 (Level S1 ~ S5)

1.  **参与团队裂变**：
    *   分享个人专属邀请码。当新用户通过您的推荐关系接入平台并部署算力设备时，您将获得全网算力调度的共振分成（计入 `teamHashpower`），并加速 R1 Token 结算。
2.  **参与交易市场**：
    *   利用价格波动。在【交易】页面，观察 R1/USDT 模拟市场的价格走势。当价格回调时市价买入 R1 积攒权益，或在价格上涨时卖出套利。

---

## 阶段四：成立“1人算力公司”与影子发行 (最终阶段)

当用户满足以下四大核心硬性门槛后，将解锁影子发行中心权限：
1.  **节点评级达标**：租赁并网设备，使节点评级升级为 **S1 共建合伙人**。
2.  **算力历史贡献**：累计 AI Token 产出 >= 500 AI Token，或达到平台规定的 AI Token 贡献等级。这只是平台贡献门槛折算，不代表设备直接产出 R1。
3.  **并网数量**：拥有至少 1 台处于运行状态的模拟 GPU 设备。
4.  **锁仓押金**：持有可支配余额 $\ge 100$ R1。

**执行影子发行**：
*   在【发行中心】填写企业名称、代币符号、初始发行价及模拟支持池筹集目标。
*   确认扣除并锁定 100 R1 押金。
*   启动影子编译器通道编译模拟合约，完成影子部署并挂牌，面向全网用户展示模拟支持池，由模拟投资者注入 USDT。

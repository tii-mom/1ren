# R1 Token 平台权益与运作机制

R1 Token 是 1人算力平台生态的底层支柱。作为平台权益通证，它贯穿于算力并网、公司挂牌、市场交易及资产清算的各个环节点。

## 1. 资产状态与区块链演进

*   **当前账本状态**：R1 目前是平台数据库及本地存储（V1 模拟环境）中的内部记账单位，用户所能看到的增减均属于平台逻辑控制。
*   **TON 链生态映射**：平台已制定明确的区块链演进计划。当全网算力规模及活跃节点数达到预设里程碑后，R1 将在 TON 区块链网络上部署为标准的 **Jetton 智能合约**，用户可将平台内积累的内部 R1 账本资产 1:1 映射兑换为 TON 链 Jetton 代币进行自由流通。

---

## 2. 核心权益与应用场景

### 2.1 影子挂牌与 100 R1 锁定押金
*   **押金要求**：为维持影子发行市场的质量，用户创建自定义算力代币时，系统会从其账户余额中扣除并锁定 **100 R1**。
*   **赎回与处置原则**：
    *   **顺利结项**：当公司影子 Token 成功募集满额，且用户算力调度计划执行完毕，锁定的 R1 押金将被退回至余额。
    *   **违规处置**：若出现断网宕机、长期不维护设备或恶意虚假宣传等行为，平台有权没收这 100 R1 押金，并将其归入系统公共清算池。

### 2.2 交易手续费折扣
*   在 R1/USDT 模拟交易市场中，默认买卖交易手续费率为 **0.3%**。
*   系统将根据用户账户内的 R1 持仓量或锁仓比例，提供阶梯式的交易手续费率折抵，最高可将手续费降低至 **0.05%**。

### 2.3 提升平台 AI Token 回收额度
*   平台每日对 AI Token (算力碎片) 的兑换回收设有系统总池上限。
*   用户的 R1 持仓量直接决定了其每日的“个人加急回收额度”。锁仓或持有更多 R1，可解锁更高的每日 AI Token $\rightarrow$ USDT 兑换变现额度。

### 2.4 设备限频与维护减免
*   物理 GPU 机房在满载运转时会产生温度堆积，需要维护与冷却。
*   长期锁仓 R1 的用户将被系统评为“核心共建合伙人”，物理机房调度系统将免除其部分模拟设备的超温降频惩罚，或以优惠价格提供冷却液等维护包。

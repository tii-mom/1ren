# R1 增长终端 V1 产品手册

欢迎使用 R1 增长终端 (R1 Growth Terminal)。本手册旨在向用户介绍「1人算力有限公司」的产品定位、核心功能以及 V1 阶段的设计规范。

## 1. 什么是「1人算力有限公司」？

「1人算力有限公司」是一家专注于去中心化人工智能算力聚合与调度的创新平台。我们相信未来的每一个独立个体都可以通过共享闲置算力、参与节点验证等方式，成立自己的“1人公司”，共同参与到全球大模型计算和推理的浪潮中。

R1 增长终端是用户接入该庞大算力调度网络的核心客户端，提供包含设备并网监控、算力产出管理、资产兑换和项目发起等多项服务。

## 2. 模拟设备与 API 权益

*   **非物理设备物理持有**：在终端内购买或租用的所有设备（例如 GPU 并网算力节点 S1 ~ S5、本地显卡共享试用节点等）均为**“模拟设备 + API 权益”**，不代表用户在现实中真实持有或接收了物理 GPU 显卡。
*   **实质权益**：用户购买设备即代表获得了由平台托管的物理算力集群的使用权分成，或等额的大模型 API 调用折扣包。设备状态、温度、损耗和加速等功能，均是用于对物理机房运行状态的模拟交互设计。

## 3. 充值与提现体系 (TON/USDT)

*   **支付媒介**：系统使用 USDT 作为唯一的模拟和测试计价单位。
*   **TON 链接入规划**：在未来版本中，平台将接入 **TON 链的 TON-USDT**，以实现原生、极速的充值与提现交互。
*   **V1 阶段规则**：
    *   **充值**：第一阶段采用链上地址监控，充值后系统自动识别交易哈希并在账户上分。
    *   **提现**：为确保资金安全性，平台对所有提现申请采用**人工多签审核**模式，拒绝一切瞬时恶意高频套利提现，审核周期一般为 24 小时。
*   **模拟环境提醒**：当前 V1 客户端为纯本地模拟测试环境，所有的 USDT 与 R1 均为测试币，不支持真实出入金。

> 2. **不承诺任何每日收益**：AI Token 产出受到全网实际任务调度频率影响，不保证每日产生恒定收益。
